# I Wanna Be Himawari

にじさんじゲーマーズの本間ひまわりさんをモチーフにした I Wanna Be The Guy 風のゲームです。

1画面分のステージしかなく、私には高度なギミックやボス戦などを作る技量と時間はありません。どなたでも勝手にステージを追加・改良して公開してくださって構いません。

**[ブラウザでプレイ (GitHub Pages)](https://himajin-no-tameiki.github.io/v2/)**
