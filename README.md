# I Wanna Be Himawari

にじさんじゲーマーズの本間ひまわりさんをモチーフにした I Wanna Be The Guy 風のゲームです。

ボリュームがなく、私には高度なギミックやボス戦などを作る技量と時間はありません。どなたでも勝手にステージを追加・改良して公開してくださって構いません。

## 環境
Unity 2018.2.6f1
